        <h2>Overview</h2>
        <p>This project is a Wikipedia scraper built using Python in Google Colab. It extracts and processes information from Wikipedia pages using the <code>wikipedia-api</code> and <code>BeautifulSoup</code> libraries.</p>
        
        <h2>Features</h2>
        <ul>
            <li>Fetch Wikipedia articles based on search queries</li>
            <li>Extract the summary, sections, and links from Wikipedia pages</li>
            <li>Store scraped data in structured formats (CSV, JSON, etc.)</li>
            <li>Easy-to-use Google Colab notebook</li>
        </ul>
        
        <h2>Usage</h2>
        <ul>
            <li>Open the Google Colab notebook.</li>
            <li>Enter the topic you want to search for.</li>
            <li>Run the scraper to fetch Wikipedia content.</li>
            <li>Process and export data as needed.</li>
        </ul>
